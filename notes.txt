  Mathematical Derivation

  The closed-form PageRank with teleportation probability p:

  G = (1-p)·Â + (p/N)·ee^T

  PageRank: r = G·r, ||r||₁ = 1

  Substituting and using e^T·r = 1:
    [I - (1-p)·Â]·r = (p/N)·e

  CLOSED FORM:
    r = (p/N)·[I - (1-p)·Â]⁻¹·e

  The analytical solver uses scipy sparse LU factorisation to solve this exactly (for N ≤ 50k). For
  larger graphs, power iteration is used. Agreement is at machine precision (L1 error ≈ 8×10⁻¹³).

  ┌────────┬───────────────────────────────────────────────────────────┐

  Stochastic Validity Checks

  - sum(r) == 1.0 — must be a valid probability distribution
  - r[i] >= 0 for all i — no negative scores
  - r[i] >= p/N for all i — teleportation guarantees a floor value

  ---
  Key design principle: PageRank is evaluated both numerically (does the math converge to the right
  answer?) and structurally (does the distribution have the right heavy-tail shape?). GraphRAG is
  evaluated as an information retrieval problem — the PPR scores are treated like a ranked list and
  judged against a manually-curated ground truth of relevant entities.
